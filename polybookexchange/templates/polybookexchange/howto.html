{% trans "What is the book exchange ?" %}

{% blocktrans %} The book exchange is a system who allow you to sell or buy used books of others students. AGEPoly takes care of the storage of your books and sell them for you, without any fees. {% endblocktrans %}

{% trans "How the book exchange works ?" %}

{% trans "The system is very simple:" %}

{% trans "To sell a book" %}

  • {% trans "You click on 'Propose a new book' in the left menu" %}.
  • {% trans "You fill required informations and you print the generated page." %}
  • {% trans "You bring your book, with the printed page at the AGEPoly's desk." %}
  • {% trans "We put your book in the system." %}
  • {% trans "You get an automatic confirmation on your @epfl.ch address." %}
  • {% trans "When your book has been sold by another student, you will also recive an automatic email" %}
  • {% trans "At least, you can comme back to the AGEPoly's desk, with you student card (Camipro) to collect your money" %}.

{% trans "To buy a book" %}

  • {% trans "You browse this website to find a book" %}
  • {% trans "You memorise the number of the book (NOT THE ISBN NUMBER !)" %}
  • {% trans "If you found what you needed, you can come to the AGEPOly's desk with the money" %}
  • {% trans "You now have a new book !" %}

{% trans "Conditions" %}

  • {% trans "The seller must be an EPFL student" %}
  • {% trans "The AGEPoly won't be responsible for any lost or stolen book" %}
  • {% trans "The AGEPoly won't be responsible for any deterioration of a book" %}
